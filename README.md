

# Trabajo Integrador: Árboles con Listas en Python

En este repositorio se encuentran los siguientes archivos y carpetas:

- **Informe_Tecnico.pdf**  
  El informe técnico completo del trabajo integrador.

- **trabajo_integrador2.py**
